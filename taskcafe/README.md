# Home Assistant Add-on: Task Cafe

A local-only multi-user task management board

## Server Backend

* Backend: [ngoduykhanh/wireguard-ui][ngoduykhanh/wireguard-ui]
* See startup log for build and patch notes.

[ngoduykhanh/wireguard-ui]: https://github.com/ngoduykhanh/wireguard-ui

## Features

- Create KanBan-style task lists.
- Private and group shared lists.
- Create username to be the same as home assistant and you'll be automatically
  logged in.

> **Note:** Initial install can take a few minutes since Task Cafe is compiled
> from source.
